Método de Gauss-Seidel

El método de Gauss-Seidel es un algoritmo iterativo utilizado para resolver sistemas de ecuaciones lineales. Es especialmente útil cuando se trabaja con matrices dispersas o de gran tamaño.

Fórmula Iterativa

Para un sistema lineal representado como $Ax = b$, la fórmula para actualizar cada componente $x_i$ es:

$$ x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j=1}^{i-1} a_{ij}x_j^{(k+1)} - \sum_{j=i+1}^{n} a_{ij}x_j^{(k)} \right) $$

Pasos del Método

  1. Elegir una estimación inicial $x^{(0)}$.
  2. Actualizar cada variable $x_i$ usando la fórmula iterativa.
  3. Repetir hasta que el cambio entre iteraciones sea menor que una tolerancia dada.

Ejemplo

Resolver el sistema:

\begin{aligned} 4x + y + z &= 7 \\ x + 3y + z &= -8 \\ x + y + 5z &= 6 \end{aligned}

Con una estimación inicial de $x^{(0)} = (0, 0, 0)$, y una tolerancia de $10^{-4}$, el método iterativo nos lleva a una solución aproximada.

Ventajas

  • No requiere factorización de matrices.
  • Fácil de implementar.
  • Rápido para sistemas grandes y bien condicionados.

Condición de Convergencia

El método converge si la matriz es diagonalmente dominante o si el espectro de la matriz de iteración está dentro del círculo unitario.

Una matriz $A$ es diagonalmente dominante si:

$$|a_{ii}| > \sum_{j \ne i} |a_{ij}| \quad \text{para todo } i$$

Conclusión

Gauss-Seidel es una herramienta poderosa para la resolución numérica de sistemas lineales, especialmente en ingeniería y simulaciones computacionales.
